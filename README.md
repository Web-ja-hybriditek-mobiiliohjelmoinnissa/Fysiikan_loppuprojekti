# Sovelluksen käynnistäminen

## Tarvittavat paketit (asennus terminaalissa)

pip install -r requirements.txt

## Sovelluksen ajaminen projektikansiossa

Aja sovellus projektin juurihakemistossa komennolla:

python -m streamlit run app.py

## Sovelluksen ajaminen suoraan GitHubista

Sovellus voidaan ajaa suoraan GitHub-reposta ilman projektin kloonaamista:

streamlit run https://raw.githubusercontent.com/Web-ja-hybriditek-mobiiliohjelmoinnissa/Fysiikan_loppuprojekti/main/app.py
